# The Chrono-Rancher

## What is this game?
The Chrono-Rancher is a puzzle game about herding critters into time loops, originally created for the GMTK 2025 Game Jam.

## Where is the options page?
Currently, if you have the apworld for chronorancher installed, you can use the options creator in the Archipelago Launcher to create your options.
You can view details about the options for this game on the [options subpage](options_en.md).

## What does randomization do to this game?
Each critter in the game is randomized into the item pool and will not appear in levels until it is unlocked.
if enabled, each non-tutorial level will also be locked behind an item you need to find.
Each level you complete will be a check to send out an item.
